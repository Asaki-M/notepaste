### 快捷剪切板 （开发阶段）
基于electron开发的一款桌面应用。用于快速复制粘贴代码片段/文本片段。  
代码第一阶段用AI写了一部分，目前在逐步优化代码。

#### 基本使用 
目前唤醒的快捷键 是 `ctrl + alt + k`，唤醒搜索框。  

搜索框支持两种模式：  
1. 搜索模式：在搜索框中输入关键词，可以搜索到对应的代码片段/文本片段。  
2. 命令模式：在搜索框中输入`/`，可以搜索到对应的命令。  
   - 命令列表：
     - `/config`：打开配置页面
     - `/export`：导出数据库内容（csv格式）

配置页面可以管理代码片段/文本片段。
